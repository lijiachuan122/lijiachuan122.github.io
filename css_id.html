南京理工大学学号：

 

经济管理学院学院简介

南京理工大学经济管理学院前身是1980年2月9日由原第五机械工业部在华东工程学院（南京理工大学前身）组建的企业领导干部进修系，翌年更名为管理工程系，1991年7月组建成立了经济管理学院。

学院现设管理科学与工程系、企业管理系、会计学系、国际贸易学系、应用经济学系和信息管理系等6个系，拥有管理科学与工程一级学科博士后流动站和博士学位授权点，应用经济学、管理科学与工程、工商管理、图书情报与档案管理4个一级学科硕士点，拥有高级管理人员工商管理硕士（EMBA）、工商管理硕士（MBA）、会计硕士（MPAcc）、管理工程（ MEM）、国际商务、金融硕士、图书情报档案硕士等7个专业硕士学位授权点和金融学、国际经济与贸易学、会计学、工商管理、人力资源管理、信息管理与信息系统等6个本科专业。

信息管理与信息系统专业介绍

本专业始建于1985年，2005年获得'江苏省特色专业'称号，2012年入选工业和信息化部'十二五'重点建设专业。本专业拥有工信部研究型教学创新团队和'教学、科研训练、竞赛指导'的全方位立体化培养体系。本专业毕业生遍布国防科技信息研究单位、政府、金融以及相关大型国有企业和上市公司，主要从事信息资源建设与管理、信息系统开发与管理、数据处理与挖掘、数据产品设计与开发、信息分析与预测等工作。

本专业依托于理工科院校的综合优势，以'质量第一、凝练特色、培育优势、服务社会'为指导思想，坚持以国家经济建设需求为导向和以学生为中心，着重知识教学与学生能力及素质培养的紧密结合，整合计算机技术、经济管理、科技信息管理、管理信息系统四大知识系列的课程内容，培养具有计算机网络技术应用技能、经济管理与企业信息管理基础理论、信息分析与决策基本方法、信息系统规划与管理基础理论和应用技能的现代综合型、高素质的信息管理专门人才，并重点突出学生的信息系统开发应用以及信息分析基本能力培养的专业特色。

 本专业学生重点需要掌握基于网络的信息资源管理的知识与技能、信息系统开发技术与工具、数据建模与信息分析理论及方法；需要掌握大数据采集与处理的基本方法，并将相关思想与理论应用到实践工作中；需要掌握经济管理相关知识，以为从事各类信息管理工作开展奠定基础
